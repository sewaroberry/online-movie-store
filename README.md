# Movie Shop Website

A responsive and dynamic movie shop web application where users can browse and purchase movies.

## Features
- Responsive design for mobile, tablet, and desktop screens
- Movie listing and details pages
- Backend functionality using PHP
- Database integration with MySQL (phpMyAdmin)
- Focus on clean UI and user-friendly design

## Technologies Used
- HTML
- CSS
- PHP
- MySQL (phpMyAdmin)

## Purpose
This project was built to practice web design, responsiveness, and backend integration using PHP and MySQL.

## How to Run
1. Clone the repository
2. Place the project in your local server directory (e.g. `htdocs` for XAMPP)
3. Import the database using phpMyAdmin
4. Run the project on `localhost`
